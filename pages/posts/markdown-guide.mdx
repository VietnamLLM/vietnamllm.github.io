# Getting Started with Markdown

This post demonstrates various markdown features available in Nextra.

## Text Formatting

You can make text **bold**, *italic*, or ***both***. You can also use ~~strikethrough~~.

## Code Blocks

Inline code: `npm install nextra`

Block code with syntax highlighting:

```javascript
function greet(name) {
  return `Hello, ${name}!`
}

console.log(greet('World'))
```

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(10))
```
